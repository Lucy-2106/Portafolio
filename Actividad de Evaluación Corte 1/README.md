# Proyecto 1: Sistema KLIK – Administración Inteligente de Inventarios

## Descripción

KLIK es una plataforma web diseñada para optimizar el control y seguimiento de productos dentro de un inventario de gran escala. El sistema fue desarrollado utilizando PHP y emplea archivos de texto como mecanismo de almacenamiento, permitiendo gestionar miles de registros de manera organizada y eficiente sin depender de un sistema gestor de bases de datos tradicional.

## Funcionalidades Destacadas

### Gestión de Usuarios

Incorpora un mecanismo de autenticación que restringe el acceso a usuarios autorizados, garantizando la seguridad de la información mediante el uso de sesiones activas.

### Carga Automática de Información

Dispone de un módulo encargado de generar registros de productos de forma masiva, asignando automáticamente características y niveles de existencia para facilitar las pruebas y simulaciones del sistema.

### Administración de Productos

Permite consultar, modificar, actualizar y eliminar registros almacenados, además de realizar búsquedas específicas para localizar información de manera rápida.

### Elaboración de Reportes

Integra herramientas para la creación de documentos en formato PDF, proporcionando resúmenes detallados del inventario y reportes clasificados según la disponibilidad de los productos.

## Herramientas y Tecnologías Empleadas

* **Lenguaje de programación:** PHP
* **Estructura de interfaz:** HTML5
* **Diseño visual:** CSS3
* **Generación de documentos:** FPDF
* **Almacenamiento de datos:** Archivos de texto plano

## Guía de Uso

1. Ejecutar el módulo de inicialización para generar la información base del inventario.
2. Ingresar al sistema mediante la pantalla de autenticación.
3. Acceder al panel principal para visualizar y administrar los productos registrados.
4. Utilizar las opciones de filtrado y búsqueda para localizar información específica.
5. Generar reportes en PDF según las necesidades de consulta o análisis.
